Github Pages 連接 Godaddy 網域設定

攻城人生  |  RED Lin  |  2022-04-19  |  174

轉址各種失敗
成功後補個筆記

首先在 Godaddy 購買網域
購買後到 DNS 管理頁面
參考dig指令列出的4組ip

dig command

將ip及 github pages 的網址設定好

Godaddy DNS Manage

回到 github pages setting
填上在 DNS 管理頁上設定的子網域 save 即可
github 確認設定的網址無誤後會自動申請ssl憑證
約等個2-4小時就可以去勾選底下的 Enhance HTTPS 了

Github Repo Settings

使用 hexo 部落格的朋友
記得到 source 資料夾底下產生 CNAME 填上新的網址
以免重新編譯、上傳時蓋掉部落格的 CNAME 設定

hexo source folder
